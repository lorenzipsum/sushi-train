# Architecture Decision Records (ADR)

This folder captures long-lived project decisions that should not be buried in chat or PR comments.

## ADR Files

- [0001-conversation-documentation-process](./0001-conversation-documentation-process.md)

## ADR Rules

1. Use sequential numbering: `0001-...`, `0002-...`.
2. Keep status explicit: `Proposed`, `Accepted`, `Superseded`.
3. Link related conversation notes from `docs/conversations/`.
